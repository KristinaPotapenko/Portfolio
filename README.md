# 🎨 Portfolio React App

A modern and responsive **Portfolio** web application built with **React**, showcasing my projects and skills. The app features smooth animations, a slider, and a contact form — all designed to provide a seamless user experience.

---

## 🚀 Features

- 🎞️ Custom slider for project showcases  
- 📩 Contact form with email functionality powered by `@emailjs/browser`  
- ⚡ Smooth page transitions and animations using `framer-motion`  
- 🌐 Multi-page navigation with `react-router-dom`  
- 🔝 Scroll animations and smooth scrolling via `react-scroll`  
- 🛠️ Responsive design with SCSS styling for clean and maintainable code

---

## 🌿 Branches

- All development happens in the `development` branch  
- The `main` branch contains the production-ready build files for deployment  

---
## 💻 Technologies Used

- React (JSX, Create React App)  
- SCSS for styling  
- JavaScript (ES6+)  
- `@emailjs/browser` — for sending emails directly from the browser  
- `axios` — for HTTP requests  
- `framer-motion` — animation library  
- `react-router-dom` — routing library  
- `react-scroll` — smooth scrolling between sections
